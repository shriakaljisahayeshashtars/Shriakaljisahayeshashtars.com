# Shri Akal Ji Sahaye Shashtars

We build good quality Shashtars.

## Website Structure

- `index.html` - Main website file
- `style.css` - Stylesheet (included in HTML)
- `kard.jpg` - Hero background image (required)
- `placeholder.jpg` - Product image placeholder

## Getting Started

1. Add your images (`kard.jpg` and `placeholder.jpg`) to the repository
2. Update product details in the Products section
3. Deploy to GitHub Pages or your hosting provider

## Features

- Responsive design
- Dark theme with gold accents (#d4af37)
- Navigation menu
- Product showcase
- Contact footer

## Customization

Edit `index.html` to:
- Update product information
- Add contact details
- Modify colors and styling
- Add more sections as needed
